El codigo utiliza Image.network para poder cargar imagenes a partir de url
Ajusta la imagenes para ocupar todo el ancho de la pantalla del dispositivo
Se agrega el texto de cada imagen separado gracias a un sizedbox
Se utiliza el widget elevatedButton se utiliza para poder redirigir a la pagina de ulagos
el cual recibe dos parametros la accion que realizara cuando se haga click en este, el cual en nuestro caso 
nos redigira a la pagina de ulagos, y el otro parametro es el contenido del boton, el cual le pasamos un texto.
